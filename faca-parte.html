O que é um webring?

Um webring é uma coleção de websites interligados, onde cada site é ligado ao próximo através de um sistema de navegação comum. Esses webrings permitem aos visitantes navegar de um site para outro dentro do mesmo tópicos ou interesses, facilitando a descoberta de conteúdo relacionado, ajudando a organizar a web de forma mais coletiva.

Quem pode participar?

Você pode se juntar ao webring se…

  • Você for brasileiro
  • Escrever conteúdo em português do Brasil
  • Possuir um site
  • O site for seguro para as pessoas (sem pornografia, violência gráfica ou qualquer coisa que cause constrangimento ao ser visualizado em público)
  • Seu site não é homofóbico, transfóbico, racista, sexista ou preconceituoso

Se você atender a essas condições, em geral, está apto a participar. No entanto, reservamos o direito de negar seu pedido de adesão se considerarmos que sua submissão não está adequada. Seu site pode ser sobre design, desenvolvimento, acessibilidade, negócios, psicologia, etc, contanto que seja sobre acessibilidade!

Fica vetada

A particiação de empresas de acessibilidade e organizações profissionais, já que este webring é focado nas pessoas. Consulte nossas diretrizes a para obter mais informações sobre comportamentos aceitáveis e inaceitáveis.

Como participar?

Vá para o repositório do A11yBR Github e abra um pull request.
Siga estas instruções:

  1. Faça o fork do repositório e clone seu garfo para sua máquina local.
  2. No arquivo , adicione sua entrada. Você precisará adicionar o seguinte: src/data/members.json
    • nome: o título do seu site
    • link: o URL do seu site
    • papel: descriva como você se auto intitula profissionalmente
    • imagem: o URL de uma imagem que representa você
    • alt: descrição da imagem (máximo 80 caracteres)
    • bio: Adcione uma pequena biogragfia (máximo 120 caracteres)
    • feeds: opcional você pode adicionar uma lista de feeds RSS, mas se adicionar, você precisará acrescenter as informações:
      • name: o nome para este feed RSS.
      • url: a url para este feed RSS.
    • Este é um modelo do src/data/members.json que você precisa acrescenter
